0 評論

 
霸凌防治法規即將正式上路，HR 最常問我的問題是：

「主管到底要怎麼講話，才不會踩線？」

但真正的解方從來不是直接模仿某一種語氣或話術的標準答案（因為沒有標準答案）；而是需要先提升關於「管理界線、協作方式、溝通行為風險」的人際互動敏銳度，再練習選擇低風險的管理方式和溝通決策。
也就是說，現在這個世代的企業管理挑戰是：我們都需要比以前更努力「認識人」、「理解人」，並「重新規劃和人一起工作的方式」。
​
而這些是需要花時間與人相處、對話和交流，才能學會的事情。

【職場溝通】從不法侵害談溝通風險，我們需要先了解「人」

4/9/2026

0 評論

 
這兩年，我在許多企業/組織裡陪伴管理者、主管與員工，談「友善職場溝通」與「預防不法侵害」。

我們談玩笑會不會變成冒犯、談說話的聲音語氣會不會變成壓迫、談高壓管理會不會變成霸凌。

隨著法規調整，企業/組織也開始更重視申訴機制與法律遵循，而在這些真實的現場裡，我愈來愈清楚一件事情：

「預防職場不法侵害的起點，不是法律，而是人。」

在職場裡，溝通從來不只是說話技巧的展示。它是一種「選擇」：每個人在任務壓力裡、在多線的人際關係裡、在管理權責裡做出的選擇。

我們為什麼會選擇用冒犯、威脅、冷落、孤立、侮辱或不當言詞來溝通？
我們是否知道這些選擇會帶來什麼風險？
更重要的是：我們知道自己其實可以有別的選擇嗎？

【自我成長】害怕被看見，是溝通最深的恐懼；接納自己，是表達最深的自由

3/20/2026

0 評論

 
在教學的旅程裡，我遇過許多學員。大概是看到我在舞台上呈現的樣子，大家想知道如何練出那樣的聲音變化、口條、台風和互動節奏；也想知道，該怎麼克服自己對表達和溝通的恐懼。

而我想分享的一個真相是：大家看到的這個我，並不是因為去學了什麼技巧才成為現在的樣貌，而是透過大量閱讀、思考、真實互動，以及一次次的挫折累積，慢慢學會了看見自己、理解自己，再學習調整表達。

在我的生命經驗裡，也曾經害怕說話、曾經不被允許說話、曾經不被接受想法，而我一步一步克服對說話和溝通的恐懼的方法是：學習真正地接納自己。

***
