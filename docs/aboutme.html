專注於協助組織建立成熟的溝通文化、降低衝突風險、提升心理安全， 以跨界視角拆解複雜的人際與組織互動，讓職場溝通變得可理解，讓衝突和爭議風險變得可預防、可管理。

三大跨界專業基底

法律 x 風險意識｜
台大法律系畢，擅長從制度、界線與組織責任的角度分析溝通風險， 協助企業在高風險情境中做出穩健、安全的員工關係決策與制度設計（如預防職場霸凌與性騷擾）。

媒體 x 溝通語言｜
台大新聞所畢，曾任政治新聞記者，並具有20 年活動主持與現場溝通實務經驗。 熟悉敘事、訊息、媒介傳播與人際互動的影響力，能將抽象的溝通本質轉化為清晰的理解框架。

公關 x HR x 永續｜
具有企業公關、人資管理與永續規劃實務經驗， 擅長從組織制度與文化、心理安全與治理視角整合人、組織、職場關係與溝通。

專業定位：個人職能發展 x 組織溝通治理 x 零衝突方法論

以跨界背景與整合思維設計以人為本的「CCOS 企業溝通營運系統 」，協助企業將營運策略轉化為可預期、可落實的內部管理與溝通方式。

專注於語言、行為、職場人際關係、文化與永續策略的整合，將以下抽象管理概念，轉化為組織日常可執行、可應用的溝通行為與管理節奏：
  • 團隊協作
  • 領導與管理界線
  • 友善文化(預防職場霸凌/性騷擾)
  • 多元共融(DEI)
  • 心理安全
  • 人權與永續(S+G)
